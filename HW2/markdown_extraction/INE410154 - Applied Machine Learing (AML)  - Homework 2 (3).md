## INE410154 - Applied Machine Learning

Graduate Program in Computer Science (PPGCC) - Federal University of Santa Catarina Prof. Jônata Tyska

Semester: 2024.1

## HOMEWORK 2

For this homework, you will exercise the visualization and preprocessing stages of a Machine Learning Project.

Using a dataset related to your final project:

- -create interesting questions related to the objective you aim to achieve with your project and try to answer them and find insights using summary statistics and visualization techniques
- -search for common data problems and techniques (hint: related works) used for data preparation on your type of data/application
- -Check for similar data problems in your data set and apply some of these techniques (cleaning/feature engineering/reduction) to your data

Students can find or reuse a data set related to their research project (master's or PhD). To find a suitable data set, the following platforms can be used: Kaggle, UCI ML Repository, Papers with Code. The data set must be documented: number of instances, features and their descriptions, target variable(s), data source etc.

After completing the analyses, students must present their findings in a report via Moodle.

## Deliverables:

1. Report in PDF format, following the IEEE or ACM conference format
2. Source code of the implementations, either via direct upload of GitHub URL
3. If possible, the data set used in the analyses (or a fake sample if the data is protected)
4. A presentation with at most ten slides. Some projects will be presented during the synchronous activities.
